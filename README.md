# 🏭 비지도 학습 기반 사출 공정 이상 탐지 및 제어 임계치 최적화 연구

김민성, 김제민, 이하일

## 1. 프로젝트 개요
본 연구는 실제 품질 데이터(Label)가 부재한 사출 성형 공정 환경에서 통계적 기법과 머신러닝을 활용하여 공정의 이상 징후를 선제적으로 탐지하고, 현장에서 즉시 적용 가능한 제어 임계치를 도출하는 것을 목표로 합니다.

단순한 불량 판별을 넘어, 불량을 유발하는 근본 원인(Root Cause)을 물리적 메커니즘 관점에서 규명하고 설명 가능한 AI(XAI) 모델을 통해 정량적인 관리 기준을 제시합니다.

## 2. 연구 배경
- **데이터 한계:** 제조 현장의 데이터는 대부분 라벨(양/불 판정)이 없는 비지도 형태의 시계열 데이터임. 
- **기존 방식의 문제:** 숙련자의 감이나 단순 설비 알람에 의존하여 사후 조치 중심의 운영으로 인한 비용 손실 발생. 
- **필요성:** 데이터 기반의 예지보전 시스템을 구축하여 공정 최적화 및 수율 향상 필요. 

## 3. 데이터셋 및 전처리
- **데이터 출처:** KAMP(Korea AI Manufacturing Platform) 사출 성형 데이터 ]
- **주요 변수:** `Machine_Cycle_Time`, `VP_Press`, `Barrel_Temp` 등 설비 센서 데이터 
- **가상 라벨링(Pseudo-labeling):**
  - 라벨 부재 문제를 해결하기 위해 IQR 및 군집 분석을 활용.
  - 통계적으로 유의미하게 분포를 벗어난 Lot을 '비정상(Abnormal)'으로 정의하고 `PassOrFail` 라벨 생성. 

## 4. 분석 방법론
### 4.1 핵심 인자 도출 (Effect Size Analysis)
- 대규모 데이터셋(약 100만 건)의 특성상 p-value에 의존한 검정은 한계가 있음(대규모 표본의 함정). 
- 두 집단(정상/비정상) 간의 실질적 차이를 확인하기 위해 **Cohen's d (효과 크기)** 지표 활용. 
- **분석 결과:**
  - `Barrel_Temp` (배럴 온도) 관련 변수의 Cohen's d 값이 평균 8.68로 압도적 영향력 확인.
  - 반면, `Injection_Peak_Press` (사출 압력)은 -1.52로 상대적으로 낮은 영향력을 보임. 
### 4.2 물리적 인과 메커니즘 규명
- *상관관계 분석:** 배럴 온도와 사출 압력 간의 강한 음의 상관관계 확인. 
- **메커니즘:** 배럴 과열 → 수지 점도 하락 → 유동성 증가(마찰 저항 감소) → 사출 압력 저하.
- **결론:** 압력 저하는 설비 문제가 아니라 온도 과열에 따른 결과임. 

### 4.3 제어 임계치 탐색 (Decision Tree)
- 설명 가능한 모델인 의사결정 나무(Decision Tree)를 사용하여 불량 판별 규칙 추출. 
- **도출된 임계치:** `Barrel_Temp_Z2` > **216.8℃**
- 해당 온도를 초과할 경우 불량 발생 확률이 급증하는 '티핑 포인트'로 식별됨. 

## 5. 최종 결론 및 기대 효과
- **공정 최적화 가이드:** - 사출 압력이 낮다고 강제로 압력을 높이는 것은 무의미함.
  - 배럴 온도 216.8℃를 상한 관리 기준(UCL)으로 설정하고 인터락(Interlock) 시스템 도입 제안. 
- **기대 효과:**
  - 불필요한 원재료 낭비 및 재가공 공수 절감.
  - 데이터 기반의 명확한 관리 기준으로 설비 운영 효율화.
